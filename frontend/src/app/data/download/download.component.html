Download data

You can download all data from the Lettercraft project for your own analysis. This includes all sources and data on epistolary episodes. You can download the data as a JSON file or a DOCX file.

The JSON file is most suitable for quantitative analysis, for instance with programming languages like Python or R.

Download data (JSON)

The DOCX file is most suitable if you want to read or annotate the data in programs like Microsoft Word or Nvivo.
