# Lettercraft dataset

This dataset was created as part of the project *Lettercraft and Epistolary Performance in Medieval Europe*, an NWO-Vidi funded project based in the Department of Languages, Literature, and Culture at Utrecht University.

For more information about the project, visit the [Lettercraft website](https://lettercraft.hum.uu.nl). Here you can also browse the latest version of the data.

## About the project

The Lettercraft project investigates how the early medieval societies of the Merovingian Kingdoms used the letter and asks who these users were. The project team approaches the letter as a performative medium that was read out aloud, translated and circulated in public. We explore how the letter established lines of communication between diverse social groups and across political boundaries and language-barriers. This made it an essential tool for conflict-resolution and consensus-building in a world with rudimentary infrastructure and limited public order.

Our database enables us to record and explore who these users were and what they did. We record the presence of a broad range of actors, from anonymous secretaries taking dictation to invalid seeking healing by touching the letter of a saint; and a broad range of actions, including sending, gift-giving, and reading aloud. From this, we develop new understanding of how the letter was used in early medieval Europe.

## Contents

`data.json` contains the dataset as a single JSON file. This format is suitable for programming languages like Python or R, or to use in web applications. `data.schema.json` provides a technical specification of the data format, using the [JSON schema](https://json-schema.org/) convention.

`data.docx` presents the dataset in a DOCX file, which is more suitable for human readers. This file contains the same data as the JSON file, but does not use contain unique identifiers for objects.

`CITATION.cff` contains metadata for citation. This uses the [citation file format](https://citation-file-format.github.io/).
